Web speech API demo: hands free recipes

This demo reads out steps in a recipe using the speech synthesis API, and allows the user to navigate steps with web voice recognition. This means they can cook a recipe from their phone without getting it dirty.

This demo requires a microphone, speakers, and a HTTPS connection. It may need permission to listen to your voice.

Output

Step three

Chop the chorizo into chunks roughly 1cm wide

Next, continue
Move ahead one step
Back, previous
